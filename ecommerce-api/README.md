# ecommerce-api
Ecommerce API with NodeJS

Use:
* NodeJS
    * Express
    * Sequelize
    * Swagger
* MySQL

### Load Database

Run SQL file at `database/ecommerce.sql` to MySQL Server Local.

It will create new database name `ecommerce`.

### Run:

Copy `.env.example` to `.env.local` then create the correct value in the file.

```shell
nodemon app.js
```
